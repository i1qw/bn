# 币安交易机器人使用说明

## 机器人说明

这是一个币安U本位合约中高频趋势跟踪策略。

## 1. 环境准备

- 安装Python 3.8+
- 安装所需依赖：`pip install python-binance pandas requests`

## 2. 配置说明

1. 编辑 `config.py` 文件，填入您的币安API密钥：
   ```python
   API_KEY = '您的币安API密钥'
   API_SECRET = '您的币安API密钥'
   ```

2. 根据您的风险偏好调整交易参数：
   - `LONG_LEVERAGE` 和 `SHORT_LEVERAGE`：杠杆倍数
   - `LONG_AMOUNT` 和 `SHORT_AMOUNT`：交易保证金金额
   - `exclude_symbols`：排除的交易对列表（**重要说明：这个系统适合山寨币，不适合主流币**）

## 3. 使用方法

1. 运行主交易策略：`python binance_main.py`
2. 运行止盈监控：`python binance_take_profit.py`
3. 或使用批处理文件：双击 `run_binance_bot.bat`
